# 🌍 Language Translation Tool (Python Project)

## 📌 Overview
The Language Translation Tool is a simple Python-based application that translates text from one language to another using the Google Translate API.  
It supports multiple languages and is designed as a beginner-friendly command-line project.

---

## 🚀 Features
- 🌐 Translate text into multiple languages
- 🔍 Auto-detects input language
- 💻 Simple command-line interface
- ⚡ Fast and lightweight
- 🧠 Beginner-friendly Python project

---

## 🛠️ Technologies Used
- Python 3
- googletrans library (Google Translate API wrapper)
